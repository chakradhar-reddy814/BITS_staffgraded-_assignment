# Q1 Explanation

- `whoami` verified the active Linux user account in the environment.
- `groups` listed the user’s associated groups, which helps explain the access rights available in the shell.
- `echo "$SHELL"` identified the login shell currently being used for the session.
- `pwd` confirmed the current working directory where the commands were executed.
- `ls -ls` showed the workspace layout and directory contents.
- The curl test confirmed that external network access was available to YouTube.
